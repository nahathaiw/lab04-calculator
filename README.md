# Lab04 - Calculator

A front-end calculator exercise with a styled keypad and display. The starter interface includes digits, decimal input, parentheses, clear, addition, subtraction, multiplication, division, and equals controls.

## Learning goals

- Connect HTML button and keyboard events to JavaScript handlers
- Build and update a calculator expression
- Convert display symbols such as `×` and `÷` into executable operations
- Evaluate expressions and handle clear/error states
- Style an application-like interface with CSS

## Technology

- HTML
- CSS
- JavaScript

## Run locally

Open `Lab_04 Calculator.html` in a browser.

## Status

The calculator interface and styling are present, but the event handlers and calculation logic remain TODOs in the current version.
